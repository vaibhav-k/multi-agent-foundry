# Multi-Factor Authentication Setup Guide

## Purpose

Multi-factor authentication (MFA) provides additional protection for corporate accounts.

## Enrollment Steps

1. Sign in to the company identity portal.
2. Open Security Settings.
3. Select MFA Enrollment.
4. Register the Microsoft Authenticator application.
5. Complete verification.

## Lost Phone

If your registered MFA device is lost:

- Contact IT Support.
- Request MFA reset verification.
- Register a new authentication device.
